This is a Leap Motion Client that uses the Leap Python API.
Installation:
 Get the Latest LeapMotionSDK for ubuntu from here:
 Follow the installation steps given in the SDK

Connect the LeapMotionController to the PC.
Run the Visualizer to make sure the Controller is working right.

Run the blender client first and the BlenderHMC server.

Then Run LeapClient.py in terminal.
